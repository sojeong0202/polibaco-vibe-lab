# 🌏 S-Lab 폴리바코 바이브 코딩 실습

> 성공회대학교 정치외교학과 학회 **폴리바코(Polibaco)** S-Lab 바이브 코딩 실습 자료입니다.

---

## 📖 실습 스토리

성공회대학교 정치외교학과 2X학번 **폴리**는 지역 정치 수업에서 매번 국가를 분석할 때마다 구글, 위키피디아를 왔다갔다 하는 게 너무 불편했어. 그래서 국가별로 체제, 현 정상, 주요 이슈, 메모를 카드로 정리해두는 앱을 기획했지.

AI한테 코드를 받았는데... 막상 열어보니 실행도 안 되고, 화면도 이상하고, 기능도 제대로 안 굴러가. **폴리를 도와줘!** 🆘

---

## 📁 폴더 구조

| 폴더 | 설명 |
|---|---|
| `step1-error` | 1단계: 실행 자체가 안 되는 버전 |
| `step2-ui` | 2단계: 실행은 되지만 UI가 엉망인 버전 |
| `step3-logic` | 3단계: UI는 예쁜데 기능이 안 굴러가는 버전 |
| `step4-complete` | 4단계: 완성본 (참고용) |

---

## 🚀 실습 방법

### 파일 받기
1. 우측 상단 **Code → Download ZIP** 클릭
2. 압축 풀기
3. Antigravity에서 해당 단계 폴더 열기

### 실습 진행
- 각 단계 폴더 안의 `index.html` 파일을 브라우저에서 열어 현재 상태 확인
- Antigravity AI 에이전트에게 문제를 설명하고 고쳐달라고 요청
- 단계별로 순서대로 진행 (step1 → step2 → step3 → step4)

> 💡 **Tip:** 막히면 AI한테 "이 코드 왜 안 돼?"라고 그냥 물어봐도 돼요!
